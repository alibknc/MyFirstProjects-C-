# Yaş Bulma - Find Age

Program kullanıcıdan kişi sayısı ister. Alınan sayı kadar kişinin isim ve doğum tarihlerini ister. Daha sonra en küçük yaştaki kişinin adını ve doğum tarihini söyler.
